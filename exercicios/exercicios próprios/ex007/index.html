Principais formatações

Ao usarmos formatações, podemos dar um sentido real ao conteúdo, não somente forma.

Negrito/Destaque

Ao deixarmos algo em negrito significa que queremos dar importância e destaque para algo. Para isso podemos usar a tag <b>(Não semântica) ou a tag <strong>(Semântica)

Nesta frase temos como exemplo um termo em negrito usando a tag <B>(não semântica)

Nesta frase, nós temos um termo em destaque, não apenas em negrito

Itálico / Ênfase

Para realçar a importância de algo, também podemos dar ênfase em alguma frase, termo ou, etc usando usando a tag <I>(não semântica) ou a tag <em>(semântica)

Nesta frase, temos um termo em itálico usando a TAG <I> (não semântica).

Nesta frase, temos um termo em itálico usando a TAG <Em> (semântica)

Texto marcado

Assim como em textos impressos ou escritos a mão, também podemos querer fazer uma marcação em algo importante, e em HTML5 podemos fazer fazer isso usando a TAG <MARK.>

Ao fazer uma marcação, não poderemos editar a cor com HTML, afinal, isso se trata da forma do elemento e deve se utilizar do CSS para isso.

Texto grande e texto pequeno

Estamos criando um texto grande e um texto pequeno nesse parágrafo.

Mesmo ainda sendo possível utilizar <BIG> para deixar textos maiores, a tag não deve ser utilizada, afinal, foi descontinuada no HTML5. Invés disso, devemos usar o CSS para mudar o tamanho de algum elemento.

Ao contrário da TAG <BIG>, a Tag <small> ainda se encontra em uso porque se pode atribuir um significado a isso, não apenas tamanho.

Texto deletado

Podemos marcar um texto como excluído para indicar que ele deve ser lido, mas não considerado.

Para fazer isso usamos a TAG <del>

Texto inserido

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois.

Para isso usamos a TAG <ins>

Existe também o sublinhado com a TAG <u> (não semântica)

Texto sobrescrito

Para inserir coisas do tipo x20+3

Para isso usamos a TAG <sup>
